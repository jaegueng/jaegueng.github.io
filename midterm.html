| css란?

css는 Cascading Stylesheetd의 약자이며 HTML 같은 마크업 언어로 작성된 문서를 꾸미기 위해 사용합니다.HTML 문서의 head 요소 안에 style 요소를 넣고 CSS 코드를 적으면 해당 문서를 다양하게 꾸밀수 있습니다.

CSS에 대한 더 자세한 설명은 CSS 문서를 참고 하세요.

| 중첩된 목록

| 이미지 넣기

이미지를 넣으려면 img요소를 사용합니 다.

src 속성에는 이미지 의 경로를(URL)를 적습니 다.

이미지 요소에는 항상 alt 속성을 사용하여 대체 텍스트를 명시해주어야 합니다."alt"는 "Alternative text"의 약자로, 이미지를 표시 할 수 없을 때 이미지를 대신하여 나타낼 텍스 트를 뜻합니다.
